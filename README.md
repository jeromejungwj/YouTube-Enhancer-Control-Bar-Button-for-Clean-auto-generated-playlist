# YouTube Playlist Cleaner Button

Adds a custom button to the YouTube Enhancer control bar to clean automatically generated playlists. Automatically adjusts to the control bar below the video.

## Installation

1. **Install Enhancer for YouTube**  
   Make sure you have the [Enhancer for YouTube](https://www.mrfdev.com/enhancer-for-youtube) browser extension installed.

2. **Add the Script**  
   - Open Enhancer for YouTube settings.  
   - Go to **Advanced → Custom JavaScript** (or equivalent section).  
   - Copy and paste the content of `yt-clean-playlist-button.js` into the script editor.  
   - Save changes.

3. **Configure the Control Bar**  
   - Set the **Control Bar position** to **below the video** in Enhancer settings.  
   - The button will automatically adjust to the control bar size and position.

## Usage

1. Open any YouTube video with an automatically generated playlist.  
2. Click the **“Clean Playlist”** button on the control bar.  
3. The script will remove unwanted items from the playlist URL or provide a cleaned version.
