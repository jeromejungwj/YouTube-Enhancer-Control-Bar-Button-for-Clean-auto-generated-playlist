# 유튜브 자동 생성 재생목록 정리 버튼

YouTube Enhancer의 컨트롤 바에 자동으로 생성된 재생목록을 정리할 수 있는 커스텀 버튼을 추가합니다. 비디오 아래에 위치한 컨트롤 바에 자동으로 맞춰집니다.

## 설치 방법

1. **Enhancer for YouTube 설치**  
   [Enhancer for YouTube](https://chrome.google.com/webstore/detail/enhancer-for-youtube/ponfpcnoihfmfllpaingbgckeeldkhle) 브라우저 확장 프로그램을 설치합니다.

2. **스크립트 추가**  
   - Enhancer for YouTube 설정을 엽니다.  
   - **고급 → Custom JavaScript** (또는 유사한 섹션)로 이동합니다.  
   - `yt-clean-playlist-button.js`의 내용을 복사해 스크립트 편집기에 붙여넣습니다.  
   - 변경 사항을 저장합니다.

3. **컨트롤 바 설정**  
   - Enhancer 설정에서 **컨트롤 바 위치**를 **비디오 아래**로 설정합니다.  
   - 버튼이 컨트롤 바 크기와 위치에 자동으로 맞춰집니다.

## 사용 방법

1. 자동 생성 재생목록이 있는 유튜브 영상을 엽니다.  
2. 컨트롤 바의 **“Clean Playlist”** 버튼을 클릭합니다.  
3. 스크립트가 재생목록 URL에서 원치 않는 항목을 제거하거나 정리된 버전을 제공합니다.
